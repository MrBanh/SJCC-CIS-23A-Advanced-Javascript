Bubble Sort

Enter an array of numbers, separated by a comma.
The output will be the array sorted in descending order using bubble sort.

Sample Array: [12, 345, 4, 546, 122, 84, 98, 64, 9, 1, 3223, 455, 23, 234, 213]
Sample Output: [3223, 546, 455, 345, 234, 213, 122, 98, 84, 64, 23, 12, 9, 4, 1]
